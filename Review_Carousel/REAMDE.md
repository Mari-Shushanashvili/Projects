# Review Carousel

An interactive review carousel application built with HTML, CSS, and JavaScript. The app allows users to browse through a list of reviews and view random reviews with a "Surprise Me" button.

## Features

- **Next/Previous Navigation**: Browse through reviews using navigation buttons.
- **Random Review**: View a random review with a "Surprise Me" button.
- **User-Friendly Design**: Clean and responsive UI with a professional look.
- **Dynamic Content**: Display user details including name, job title, and review message dynamically from a JavaScript array.

## How to Use

1. **Navigation**:
   - Click on the **Previous** or **Next** buttons to browse reviews in order.
2. **Random Review**:
   - Click on the **Surprise Me** button to display a random review.

## Code Structure

- **HTML**: Basic structure of the carousel with buttons and content placeholders.
- **CSS**: Custom styling for buttons, layout, and hover effects.
- **JavaScript**: Controls the review display logic and manages the carousel's dynamic functionality.

## Project Overview

- **Technologies**: HTML, CSS, JavaScript
- **Key Concepts**: DOM manipulation, event handling, randomization, responsive design
