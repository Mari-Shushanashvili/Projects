# Christmas Countdown

A fun and festive Christmas countdown clock that shows the time remaining until Christmas! This project displays the remaining days, hours, minutes, and seconds in a beautifully styled layout. The design incorporates a Christmas theme with custom fonts and a countdown timer.

## Features

- **Countdown Timer**: Displays the time remaining until Christmas.
- **Responsive Design**: The layout is centered and adjusts to different screen sizes.
- **Custom Fonts**: Utilizes the 'Great Vibes' cursive font for a festive look.
- **Background**: Customizable background image with full-screen coverage.

## Tech Stack

- HTML
- CSS
- JavaScript
