# Image Slider  

Welcome to the ** Image Slider**, a visually stunning and fully responsive image slider. This project combines the power of **HTML**, **CSS**, and **JavaScript** to deliver a seamless and interactive experience for users.  

---

##  Features  

###  Interactive Controls  
- **Navigation Arrows**: Hover near the edges to reveal translucent, sleek arrow buttons for manual navigation.  
- **Auto-Slide**: The slider cycles through images automatically every 3 seconds for a hands-free experience.
